# Blood Donation Social

## Project Overview
This project aims to connect blood donors and recipients through a social platform, allowing users to easily find and offer blood donations.

## Features
- User registration and profile management
- Search functionality for nearby donors and recipients
- Real-time notifications for donation requests
- Integration with maps for location services
- Community features like chats and forums

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Authentication:** JWT
- **Hosting:** Heroku or Vercel

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/blood-donation-social.git
   ```
2. Navigate to the project directory:
   ```bash
   cd blood-donation-social
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Set up environment variables as provided in the `.env.example` file.
5. Start the development server:
   ```bash
   npm start
   ```

## Usage
- After starting the server, navigate to `http://localhost:3000` to access the app.
- Sign up to create an account or log in if you already have one.
- Use the search feature to find nearby donors or recipients.
- Receive notifications for any donation requests.
